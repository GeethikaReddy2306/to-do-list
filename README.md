#📝 To-Do-List-webapp


A simple and responsive application built with **HTML, CSS, and JavaScript**.  
  This project enables users to add tasks, delete a specific task, and remove all tasks simultaneously.

---

## 🚀 Features
-Add new tasks to the list  
-Remove a specific task   
- Clear the entire list 
- Fast and smooth interactions 

 🔗 Live Demo  
👉 [Click here to view the app](https://to-do-app-frontend.netlify.app/)  


## 🛠️ Technologies Used
- **HTML5**  
- **CSS3** (Responsive styling & Flexbox)  
- **JavaScript (ES6)**

## ▶️ How to Use

1. Type your task in the input box  
2. Click **Submit** to add it to the list  
3. Click **Delete** on any task to remove it  
4. Click **Remove All** to clear the whole list
